## Introduction

This is a simple driver fatigue monitoring based on eye status.

The face detection using hdface.

The eyes keypoints detectiong using PFPLD.



## Requirements

- python 3

- hdface == 0.1.3

- numpy == 1.16.6

- opencv-python == 4.2.0.32

- torch == 1.1.0

- torchvision == 0.3.0
